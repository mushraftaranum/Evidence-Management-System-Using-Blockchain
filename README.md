# 🔗 Evidence-Management-System-Using-Blockchain

Evidence Management System (EMS) is a blockchain-based application designed to securely store, manage, and verify digital evidence using Ethereum and IPFS.

This system ensures tamper-proof storage of evidence by leveraging smart contracts and decentralized file storage. It provides role-based access control for Police and Court users, maintaining transparency and integrity in the evidence handling process.

## 🚀 Features

- 🔐 Secure evidence storage using blockchain
- 📁 File storage using IPFS (via Pinata)
- 👮 Police can upload evidence with details
- ⚖️ Court can verify and download evidence
- 🔑 Password-protected access for sensitive data
- 🦊 MetaMask authentication
- ⛓️ Tamper-proof and transparent system

## 🛠️ Technologies Used

- Solidity (Smart Contracts)
- Ethereum (Ganache - Local Blockchain)
- Web3.js
- IPFS (Pinata)
- MetaMask
- HTML, CSS, JavaScript

## ⚙️ How It Works

1. User registers as Police or Court
2. Police uploads evidence with details and password
3. File is stored in IPFS and hash is stored on blockchain
4. Court verifies evidence using ID and password
5. Evidence is securely accessed and downloaded

## 📌 Use Case

This system is useful for:
- Digital forensics
- Law enforcement agencies
- Secure legal evidence management
